# PublicHolidaysList

An `PublicHolidaysList` component can be used list PublicHolidays in a table.


export const Template = (args) => <PublicHolidaysList {...args} />

A table with ten public holidays (the ones for the US in 2021) looks like this:

<Canvas>
  <Story name="US2021" args={{ label: 'US2021' , data: publicHolidayFixtures.US2021 }}>
    {Template.bind({})}
   </Story>
</Canvas>

A table with one holiday looks like this:

<Canvas>
  <Story name="One Holiday" args={{ label: 'One Holiday' , data: publicHolidayFixtures.oneHoliday  }}>
    {Template.bind({})}
   </Story>
</Canvas>

An empty table looks like this:

<Canvas>
  <Story name="Empty" args={{ label: 'Empty' , data: []  }}>
    {Template.bind({})}
   </Story>
</Canvas>

And if you pass in null for the data, you get this:

<Canvas>
  <Story name="Null" args={{ label: 'Null', data: null }}>
    {Template.bind({})}
   </Story>
</Canvas>
