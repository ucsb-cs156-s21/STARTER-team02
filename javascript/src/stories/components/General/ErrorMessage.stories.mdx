# ErrorMessage

The `ErrorMessage` component can be used to create an error message
that has the expected CSS class (`error-message`) and a `data-testid`
value.

Here is an example usage.  
* The parameter `condition`, i.e. in this case,
  `errors.distance` is treated as a  boolean value, and the `ErrorMessage`
  component appears only if that value evaluates to true.
* The `testId` parameter should be a unique identifier that can be used
  to identify the element in test code.

```jsx
           <ErrorMessage 
              condition={errors.distance}
              text={"Distance is required and should be a decimal number"}
              testId={"distance-errors"}
            />

```
