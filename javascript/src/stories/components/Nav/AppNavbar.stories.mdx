# AppNavbar

The `AppNavbar` appears at the top of every page of the app. 

export const Template = (args) => {
  return (
    <BrowserRouter>
      <AppNavbar {...args} />
    </BrowserRouter>
  )
}

<Canvas>
  <Story name="Example" args={{ label: 'Example' }}>
    {Template.bind({})}
   </Story>

Note that the `AppNavbar` component must
appear in the context of a `Router` component such as 
`BrowserRouter` or there will be errors such as this one:

```
Error: Invariant failed: You should not use <withRouter(LinkContainer) /> outside a <Router>
```

For example:
```
<BrowserRouter>
  <AppNavbar />
</BrowserRouter>
```
