# EarthquakesPage

The `EarthquakesPage` can be used to get data from the backend API
endpoint `/api/earthquakes`.   In the storybook, since a backend is 
not running, filling out the form will result in a toast notifications
with a `404` message that the api endpoint is unavailable.
